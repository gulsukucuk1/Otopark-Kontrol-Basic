# Otopark-Kontrol-Basic
Nesneye Yönelik Programlama dersi vize projesi kapsamında geliştirilen bu uygulama, C# dili ile hazırlanmış bir Otopark Yönetim Sistemidir. Projede sınıflar, nesneler, kapsülleme ve kalıtım gibi NOP prensipleri kullanılmış; araç giriş-çıkış işlemleri ile otopark kapasitesinin takibi amaçlanmıştır.
